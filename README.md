# Hybrid BDD ThreadLocal Automation Framework

## Overview
This project is a Selenium automation framework built using Java, Cucumber BDD, TestNG, and Page Object Model.

The framework supports parallel execution using ThreadLocal WebDriver and generates Extent Reports.

## Tech Stack
- Java
- Selenium WebDriver
- Cucumber BDD
- TestNG
- Maven
- Page Object Model
- ThreadLocal (Parallel Execution)
- Extent Reports

## Framework Features
- Parallel execution
- Thread-safe WebDriver
- Config driven browser
- Screenshot capture
- BDD feature files
- Page Object Model

## Execution

Run using Maven:

mvn clean test
